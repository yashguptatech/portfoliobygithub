# Portfolio by GitHub

A dynamic portfolio website that automatically fetches and displays projects from GitHub.

## Features

- **Dynamic Project Loading**: Automatically fetches and displays repositories from GitHub API
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **GitHub Integration**: Shows real-time stats including repository count and stars
- **Skills Showcase**: Highlights technical skills across multiple domains
- **Social Links**: Direct links to GitHub and LinkedIn profiles

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub API
- Font Awesome Icons

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. The portfolio will automatically fetch and display projects from the GitHub username specified in `script.js`

## Customization

To customize this portfolio for your own use:

1. Update the `GITHUB_USERNAME` constant in `script.js` with your GitHub username
2. Update the LinkedIn URL in `index.html`
3. Modify the `FEATURED_REPOS` array in `script.js` to highlight your preferred projects
4. Customize the skills section in `index.html` to match your expertise
5. Update the profile image URL or add your own image

## GitHub Pages

This portfolio can be easily deployed to GitHub Pages:

1. Go to your repository settings
2. Navigate to the "Pages" section
3. Select the branch you want to deploy (usually `main`)
4. Save and your portfolio will be live at `https://[username].github.io/[repository-name]`
